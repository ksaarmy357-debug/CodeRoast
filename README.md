# 🔥 CodeRoast

A brutally honest AI code reviewer that roasts your code like a savage senior dev — but still teaches you something.

## Setup

1. Get a free API key from https://openrouter.ai/keys
2. Open `index.html` in your browser.
3. When you run your first roast, it will prompt you for your API key. (The key is saved in your browser's local storage securely!).

## How to use

- Paste any code into the editor
- Select the language
- Select your preferred AI Model (e.g. Claude Sonnet 4, Gemini, Nvidia Nemotron, etc.)
- Hit "Roast My Code"
- Cry, laugh, then actually fix your code

## Stack

- Plain HTML, CSS, JavaScript (no frameworks)
- OpenRouter API (Access to Claude, Llama, Gemini, DeepSeek, etc.)
